Heading Action

The Heading action changes the heading style of the text at the cursor.

The available options for this command are all of the options on this menu — "Normal text", "Subtitle", "Heading 1", "Heading 2", and "Heading 3". In addition, Headings 4-6 can be accessed if they are first used manually, after which they appear on the menu.

Note: this action is particularly unstable. Random changes to the Google Docs UI could break this.
